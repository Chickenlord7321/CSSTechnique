Intro

What You Can Learn From This Tutorial

This tutuorial covers the basics of creating a vertically-oriented gallery with only CSS Flex and Grid. Using Flex and Grid, we are able to create a gallery that works for all different sizes of images. The styling of these images, however, is an exercise left to the reader. The intention of this tutorial is to give you a foundation from which to work, while allowing you the freedom to choose how you want the design the gallery.

The video to the right goes through the process of creating a gallery in real time, explaining what the code does and how it works.

Step 1

HTML Setup

First, we create and set up an index.html file for our Responsive Gallery website. You can download the file directly if you would like, or create your own.

We populate index.html with a div called "gallery", and inside we create 4 divs to be the columns. Within those columns you can include as many img elements as you want. In this example, we are using images sourced from Lorum Picsum, a site which provides random images from their database. The images have random widths and heights between 600 and 1,000 pixels.
